# VGS 全球實時生活劇場 v7.2 (Universal Lifestyle Engine)

這是一套為 AI 圖像生成量身定做的 **「實時生活劇場分鏡系統」**。它最初為 **Goro Baseball (nanobanana)** 開發，並在 V7.2 版本中進化為一套通用的、具備好萊塢級 3D 空間感的生活流敘事模板。

## 🎬 核心特色：
- **實時時空同步 (Real-time Calibration)**：自動對齊目標城市的實時時間、天氣與溫度。
- **動態生活流 (Living Flow Protocol)**：擺脫「製服化」，根據氣溫與場合自動配備民生私服。
- **空間微粒協議 (Spatial Dust Protocol)**：在光影中部署 3D 塵埃微粒，視覺深度極致拉開。
- **三層構圖敘事 (Multi-Layered Storytelling)**：前景物、中景演技、後景生命、環境 FX。

## 📂 檔案結構：
- `SKILL.md`：核心憲法與通用 6 階 SOP。
- `db/assets.json`：全球地理資產與 FX 庫。
- `db/phenotype.json`：角色身分特徵物理鎖定。
- `DEV_MEMORY.md`：專案演進歷史與「結案日誌」。

## 🚀 如何使用：
1. **變數初始化**：在 `SKILL.md` 中定義您的 `Identity (Image 1)` 與 `Locale`。
2. **執行 SOP**：遵循 1-6 階工作流，獲取具備「空氣厚度感」的 JSON 提示詞。
