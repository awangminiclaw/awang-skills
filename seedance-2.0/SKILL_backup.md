# Seedance 2.0 Skill

## 簡介

Seedance 2.0 是一個革命性的多模態 AI 影片生成模型，支援圖像、影片、音訊和文字輸入。本技能檔案協助您生成完整的分鏡腳本，包含 9 個元素的專業電影級提示詞。

## 官方文件

- **官方網站**: https://seedance2.ai/zh-hant
- **素材限制**: 圖片最多 9 張、影片最多 3 部（總時長 ≤15秒）、音訊最多 3 個（MP3，總時長 ≤15秒）
- **總素材限制**: 最多 12 個檔案

1. **每個分鏡必須包含完整的 9 個元素**
2. **分鏡編號按時間碼順序**（鏡頭1, 鏡頭2, 鏡頭3...）
3. **角色格式**：`{性別/類型}角色{N}（@image{N}），{服裝設計}，{道具}，{動作}`
4. **場景格式**：`{場景名稱}場景{N}（@image{N}）`
5. **角色數量上限**：最多 5 個角色（分配 @image1-@image5）
6. **場景數量上限**：最多 4 個場景（分配 @image6-@image9）
7. **所有術語使用電影專業術語**（場面調度、景深、焦點轉移等）
8. **對白使用純正英文**（Authentic English Dialogue）
9. **音訊全面 Dolby Atmos 規格**
10. **善用特寫分鏡聚焦重點**（Close-Up for Key Moments）
11. **特寫眼神表演**（Eye Performance in Close-Ups）

### 特寫分鏡規範

#### 特寫鏡頭使用原則

特寫鏡頭（Close-Up, CU）與大特寫（Extreme Close-Up, ECU）是關鍵敘事工具，用於：

| 使用時機 | 景別 | 說明 |
|---------|------|------|
| **情感高潮** | CU / ECU | 角色情感爆發的瞬間 |
| **關鍵對白** | MCU / CU | 重要對話的情感傳達 |
| **眼神表演** | ECU | 眼神透露內心世界 |
| **反應鏡頭** | CU | 角色對事件的反應 |
| **道具揭示** | ECU | 關鍵物件的細節呈現 |

#### 特寫眼神表演

**眼神是角色靈魂的窗口**，大特寫眼神是電影敘事最強大的工具：

```
眼神表演層次：
├─ 凝視方向 - 角色注視何處，揭示專注點
├─ 眼神光 - 眼中的光線，反映情感狀態
├─ 瞳孔變化 - 恐懼（放大）、憤怒（收縮）
├─ 眼瞼狀態 - 緊繃（壓力）、放鬆（平靜）
├─ 眉毛配合 - 皺眉（擔憂）、揚眉（驚訝）
└─ 眼角細節 - 魚尾紋（微笑）、淚光（悲傷）
```

#### 特寫分鏡範例

#### 特寫與遠景的交替節奏

專業電影善用**特寫與遠景的交替**來控制節奏與張力：

```
鏡頭交替模式：
├─ 遠景（ELS）建立場景 → 特寫（CU）揭示情感 → 遠景（LS）環境語境
├─ 特寫（ECU）眼神 → 中景（MS）對話 → 特寫（CU）反應 → 遠景（LS）釋放
└─ 平行剪輯：特寫A ↔ 特寫B（對話場景）
```

### 音訊規範

#### Dolby Atmos 音訊架構

所有音訊必須採用 **Dolby Atmos 7.1.4** 格式，包含：

```
Dolby Atmos 7.1.4 聲道配置：
├─ 7 個地面聲道
│├─ 前左
│  ├─ 前中
│  ├─ 前右
│  ├─ 環繞左
│  └─ 環繞右
├─ 1 個低頻效果聲道
│  └─ Subwoofer (LFE)
└─ 4 個高度聲道
   ├─ 頂前左
   ├─ 頂前右
   ├─ 頂後左
   └─ 頂後右
```

#### 音訊分層設計

| 層級 | 元素 | 空間定位 | 說明 |
|------|------|---------|------|
| **Layer 1** | 環境音 | Dolby Atmos 360° | 自然環境聲、城市聲、風聲 |
| **Layer 2** | 特效音（SFX） | 動態定位 | 物理撞擊、爆炸、機械聲 |
| **Layer 3** | 對白 | 前置中央 | 純正英文對白，清晰可辨 |
| **Layer 4** | 音樂 | 環繞混音 | 主題音樂、情感配樂 |

#### 音訊提示詞範例

```
【音訊】Dolby Atmos 7.1.4：
├─ 環境音：城市車流（360度遠景），晨鳥叫聲（頂前），風聲（環繞）
├─ 特效音：腳步聲（前左→前右移動），門開啟聲（前中），裝備碰撞（前右）
├─ 對白：男角色1純正英文對白（前置中央清晰）
└─ 音樂：主題音樂漸入（環繞混音）
```

#### 純正英文對白規範

- **所有對白使用純正英文**（Authentic English）
- **避免翻譯腔**（Translationese）
- **符合角色背景**（口音、語氣、用詞）
- **對白清晰可辨**（前置中央定位）
